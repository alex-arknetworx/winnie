Framebuffer Window System

Compiles and runs on Debian 11.11 console with no X or Wayland [uses fbdev].

Clone the repo then:

cd winnie/libwinnie/
make
cd ../clock/
make
cd ../sysmon/
make
cd ../tunnel/
make
cd ../winnie/
make
sudo ./wserver

press q to quit back to console
